# 保發中心保單條款 PDF 爬蟲專案說明

本專案旨在自動化搜尋並下載財團法人保險事業發展中心（保發中心）的保險商品保單條款 PDF 檔案。

## 📁 檔案分類與功能說明

### 1. 環境診斷工具（確認網頁結構）

當保發中心網站改版或結構改變時，優先執行此分類的腳本來修正爬蟲定位。

* **`step1_diagnose_frames.py`**：診斷網頁是否含有 `frame` 或 `iframe` 頁框結構，並嗅探各層級的關鍵字與輸入框 ID。
* **`step2_diagnose_query.py`**：針對特定的查詢頁面（`Query.aspx`），全面掃描並印出所有選單（`select`）、輸入框（`input`）與按鈕的 ID，供爬蟲定位參考。
* **`read_debug.py`**：使用 BeautifulSoup 解析本地 HTML 暫存檔，用於靜態分析網頁標籤與圖形驗證碼欄位。

### 2. 進階功能爬蟲（按「險種分類」查詢）

* **`tii_health_scraper.py`** (v1)：半自動爬蟲。自動選取人身保險與特定類別，暫停等待人工輸入圖形驗證碼，隨後自動翻頁解析 PDF。
* **`tii_health_scraper_v3.py`** (v3)：優化版爬蟲。新增自動勾選「未停售」功能，並改為「先收集商品列表，再逐一進入詳細頁下載保單條款」的精準下載模式。
* **`tii_pdf_downloader.py`**：全自動分流爬蟲。針對「醫療、癌症、意外」三大險種進行批次自動切換框架與分流下載。

### 3. 全文檢索爬蟲（按「關鍵字」查詢）

* **`tii_pdf_downloader_v2.py`**：**目前最穩定、完整的版本。** 針對保發中心的「全文檢索功能」設計，自動依序輸入「醫療、實支實付、手術」等 7 大關鍵字，支援人工填寫驗證碼，並具備強大的 Requests Session 同步下載機制與翻頁功能。

### 4. 執行日誌與除錯紀錄

* **`pdf_downloader.log`**：記錄爬蟲執行過程的詳細日誌（包含成功填入、選擇資料庫、或查無資料的警告）。
* **`debug_form_page.html`** / **`debug_result_page.html`**：當爬蟲在表單頁或結果頁遇到異常時，自動抓取的網頁原始碼快照，用於人工排查錯誤。
